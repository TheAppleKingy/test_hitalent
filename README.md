# Тестовое задание для компании hitalent

**В рамках данного тестового задания было создано простое приложение для работы с чатами и сообщениями. Доступные эндпоинты:**

- POST /chats - данный эндпоинт принимает данные чата(title) и создает его.


- POST /chats/{chat_id}/messages - данный эндпоинт принимает текст сообщения(text), создает его и закрепляет за чатом(chat_id)

- GET /chats/{chat_id}. Query-параметры - limit. Данный эндпоинт возвращает данные чата по chat_id(chat) а также limit последних сообщений(messages). По умолчанию limit=20, максимальное значение - 100.

- DELETE /chats/{chat_id} - данный эндпоинт удаляет чат по chat_id.


>Все пункты раздела "Логика и ограничения" тех. задания удовлетворены.

# Quickstart
Для запуска приложения выполните в папке проекта:
```
docker compose up --build
```
>Миграции применятся автоматически
